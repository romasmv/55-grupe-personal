### 🏃 Run locally

Would like to run this project locally? Open terminal and follow these steps:

1. Clone the repo
   ```sh
   git clone https://romasmv.github.io/55-grupe-personal.git
   ```
2. Install NPM packages
   ```sh
   npm i
   ```
   or
   ```sh
   npm install
   ```
3. Run the server
   ```sh
   npm run dev
   ```
